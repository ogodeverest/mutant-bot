## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/mutant-bot.git
```

Go to the project directory

```bash
  cd mutant-bot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Client:** Three.js
